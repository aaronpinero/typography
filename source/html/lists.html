List Styles

Ordered, unordered, and definition lists inherit styles from the essential/body and essential/spacing modules.

Since lists are used in many situations and for many types of content, Tyfy offers several list classes that create variations on the default HTML lists.

Classes

.list-icons
Applied to an unordered list to replace the default bullets with icons. A second class is required to identify the icon to be used. Tyfy includes the list-icons-angle class which uses the small angle right icon. A project can define additional icon classes based on list-icons-angle.
.no-bullet
Applied to an unordered list to remove the bullets and left-hand padding.
.indented
Primarily for use with no-bullet, this class indents all lines after the first line in each list item to make it easier to identify individual items.
.list-cols-2, .list-cols-3
Applied to ordered and unordered lists to display items in columns. These classes use CSS columns; columns are read top to bottom and then left to right. These classes can be used with the no-bullet and indented classes.
.list-grid-2, .list-grid-3, .list-grid-4
Applied to ordered and unordered lists to display items in rows. These classes use CSS floats; rows are read left to right and then top to bottom. These classes can be used with the list-icons class.

Related Variables

$tyfy-font-size-base
Lists inherit this font size from the <body>.
$tyfy-color-foreground
Lists inherit this font color from the <body>.
$tyfy-spacing-standard
$tyfy-spacing-standard is the margin between lists and other elements. Inside lists, $tyfy-spacing-standard/2 is the spacing between list items.
$tyfy-indent-standard
The amount of indentation for list items, definition list definitions, and indented text using the indented class; applied using padding-left.
$tyfy-font-base
Lists inherit this font family from the <body>.
