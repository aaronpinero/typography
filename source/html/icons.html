Icons

Originally, I wanted to make FontAwesome Pro (my favorite icon font) part of Tyfy. It was one of the original dependencies, included as a .git submodule. Eventually, I realized it could be annoying to other users to require a FontAwesome Pro license. In addition, Tyfy only needs a handful of icons, so the exhaustive FontAwesome Pro library is not really needed.

Node.js and the webfont package are used to convert a set of .svg icons into an icon webfont called tyfyicons. The Sketch file used to create the .svgs is included in the repository.
