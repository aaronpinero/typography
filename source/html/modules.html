Modules

The tyfy.scss file is an inventory of the modules that make up the framework. They are @imported in the following order.

Icon Font

The Tyfy custom icon webfont appears first in the tyfy.scss file but is not required. You can use your own icon font or any of the freely available icon fonts.

@import "../../dist/fonts/tyfyicons/tyfyicons.scss"
Defines the font face, SASS variables for the icon character codes, and utility classes for the icons.

Required

These modules are required to use Tyfy. They define the framework's variables, functions, and mixins.

@import "required/core"
Default values for the core Tyfy variables; these variables are used in the calculation of many of the default settings and are required by the custom functions; see the variables documentation for more
@import "required/functions"
Functions facilitating use of the basic grid unit, modular typographic scale, and primary color
@import "required/settings"
Default values for variables that define the projects colors, fonts, breakpoints, etc.; see the variables documentation for more
@import "required/mixins"
Useful mixins taken from HTML5 Boilerplate

Essential

These modules are not hard requirements for Tyfy but are highly recommended.

@import "essential/normalize"
Selected styles from normalize.css
@import "essential/body"
Apply the base colors, font styles, and line height
@import "essential/spacing"
Apply grid-based vertical margins to HTML elements

Optional

These optional modules provide specific typographic or presentation styles.

@import "optional/paragraphs"
Styles and utility classes for paragraphs and headings
@import "optional/links"
Styles and utility classes for links
@import "optional/lists"
Styles and utility classes for lists
@import "optional/inline"
Styles and utility classes for inline-level elements
@import "optional/quotes"
Styles and utility classes for blockquotes and pullquotes
@import "optional/tables"
Styles and utility classes for tables; responsive tables requires tables.js
@import "optional/reverse"
A utility class for reversed colors
@import "optional/collapsables"
A presentation style for collapsable sections; requires collapsable.js
@import "optional/tabs"
A presentation style for responsive tabs; requires tabs.js
