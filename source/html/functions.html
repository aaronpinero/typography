Functions

Tyfy includes several custom SASS functions. These functions were written to facilitate writing the Tyfy SCSS files, but can also be used to help users who are including Tyfy in their web projects.

@function leading($font-size,$leading-style)
The leading function calculates a grid-based line height for a given font size and leading style option. The leading styles are tall, base, heading, and short. The font size provided must be in rem units, and the value returned is also in rem units.
@function grid($multiplier)
The grid function is used to make the SCSS files easier to read. The function accepts a number and returns a value, in rem units, that is the base grid unit multiplied by the number provided.
@function scale($exponent)
The scale function facilitates use of the modular typographic scale by eliminating the task of calculating font sizes. The function accepts an integer and returns a font size, in rem units, that is the specified number of steps above or below the base font size on the typographic scale. The value 0 will return the base font size. If the argument is 2, the function will return the font size that is 2 steps larger on the scale. If the argument is -1, the function will return the font size that is 1 step smaller on the scale.
@function color($color,$lightness)
The function accepts a color (in rgb or hexadecimal format) and a lightness value (0% to 100%), and converts the given color to the specified lightness.
@function reverse($color)
The function accepts a color (in rgb or hexadecimal format) and returns the same color but with the lightness value reversed.
@function charcode($code)
The function accepts a character code without the preceding slash (“\”) and adds it; this is done to get around a problem in SASS and allow icon character codes to be more easily customized for different icon fonts.
