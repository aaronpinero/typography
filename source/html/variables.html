Variables

Tyfy has a library of SASS variables that define the base grid, colors, text sizes, fonts, breakpoints, and other characteristics. My goal with this project is to have as few variables as possible. This results in some key variables – such as the grid unit, brand color and font size scalar – that can quickly change the entire character of the project to which Tyfy is applied.

Variables are organized into five files for easier reading. All variables begin with the prefix $tyfy- to avoid name collision with other projects.

variables/grid

$tyfy-grid-unit
Base grid unit in rem units; I recommend this value be half the default line height
$tyfy-leading-adjustment-tall
Unitless value used in by the leading function to calculate tall leading; values less than 1 result in larger leading
$tyfy-leading-adjustment-base
Unitless value used in by the leading function to calculate regular leading; values less than 1 result in larger leading
$tyfy-leading-adjustment-heading
Unitless value used in by the leading function to calculate heading leading; values less than 1 result in larger leading
$tyfy-leading-adjustment-short
Unitless value used in by the leading function to calculate short leading; values greater than 1 result in smaller leading
$tyfy-spacing-standard
Standard spacing between HTML elements defined as a number of grid units
$tyfy-spacing-large
Extra spacing between HTML elements; used for spacing above headings
$tyfy-indent-standard
Standard indent distance for lists and blockquotes defined as a number of grid units
$tyfy-indent-small
Small indent distance for lists and blockquotes at small screen sizes

$tyfy-mono
Monospace font selection

$tyfy-font-heading

$tyfy-font-pullquote

$tyfy-font-table
Table cell font selection

Unitless value that defines the proportion between sizes in the typographic scale

$tyfy-font-size-sm
Small font size
$tyfy-font-size-h6
Level 6 heading font size
$tyfy-font-size-h5

$tyfy-font-size-h4

$tyfy-font-size-h3

$tyfy-font-size-h2

Pullquote font size

variables/breakpoints

$tyfy-break-floats
Below this width floated elements are not floated, instead shown at 100% width
$tyfy-break-pullquote-large
Above this width the margins around pullquotes increase
$tyfy-break-list-cols-2
Above this width lists can be 2 columns
$tyfy-break-list-cols-3
Above this width lists can be 3 columns
$tyfy-break-list-cols-4
Above this width lists can be 4 columns
$tyfy-break-tabs
Below this width responsive tabs convert to collapsable sections
$tyfy-break-table
Below this width responsive tables are converted to labeled lists
