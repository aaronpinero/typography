Variables

Tyfy has a library of SASS variables that define the base grid, colors, text sizes, fonts, breakpoints, and other characteristics. My goal with this project is to have as few variables as possible. This results in some core variables – such as the grid unit, brand color and font size scalar – that can quickly change the entire character of the project to which Tyfy is applied. These core values are stored in the required/core file.

The remainder of the variables, most of which are defined as a function of a core value, are stored in the required/settings file.

All variables begin with the prefix $tyfy- to avoid name collision with other projects.

required/core

$tyfy-grid-unit
Base grid unit in rem units; I recommend this value be half the default line height
$tyfy-leading-adjustment-tall
Unitless value used in by the leading function to calculate tall leading; values less than 1 result in larger leading
$tyfy-leading-adjustment-base
Unitless value used in by the leading function to calculate regular leading; values less than 1 result in larger leading
$tyfy-leading-adjustment-heading
Unitless value used in by the leading function to calculate heading leading; values less than 1 result in larger leading
$tyfy-leading-adjustment-short
Unitless value used in by the leading function to calculate short leading; values greater than 1 result in smaller leading
$tyfy-font-size-scalar
Unitless value that defines the proportion between sizes in the typographic scale
$tyfy-font-size-base
Fonts size applied to the <body> tag
$tyfy-serif
Serif font selection
$tyfy-sans
Sans-serif font selection
$tyfy-mono
Monospace font selection
$tyfy-symbol
Symbol font; set to the Tyfy icon webfont
$tyfy-color-background
Background color of the <body> element
$tyfy-color-foreground
Color of the <body> element
$tyfy-color-primary
Your brand color
