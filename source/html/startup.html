Get Started

Overview

Tyfy is intended as a companion to your web project. It is a collection of SCSS stylesheets and jQuery-based JavaScript that provide a basic grid, typographic styles, and presentation styles. The grid and styles can be customized for your project by overriding the default SASS variable values.

I have tried to make Tyfy as modular as possible. You can use all of it or pick just the sets of styles that are right for your project. JavaScript is only required for certain presentation styles, and the JavaScript files are modular; each file is required only when using the specific associated presentation styles.

Requirements

  • A SASS compiler
  • jQuery v.1.12.4 or above
  • Normalize.css (included as a git submodule)
  • FontAwesome Pro (included as a git submodule)

These are the basic requirements for using Tyfy in your project. If you wish modify and recompile the CSS and documentation files, you will need node.js and npm. The included project.json file describes the required node packages.

Setup

  1. Add the FontAwesome Pro Core and Solid CSS files to your project. They can be @imported to your main stylesheet or added to the HTML.
    [path to tyfy]/dist/vendor/fontawesome/css/fontawesome-pro-solid.css
    [path to tyfy]/dist/vendor/fontawesome/css/fontawesome-pro-core.css
  2. Add the normalize.css file to your project. It can be @imported to your main stylesheet or added to the HTML.
    [path to tyfy]/dist/vendor/normalize/normalize.css
  3. Add Tyfy to your main .scss file.
    @import "[path to tyfy]/source/scss/tyfy.scss"
  4. Add Tyfy JavaScript to your project in the contexts where it is appropriate.
    • [path to tyfy]/dist/script/collapsables.js for collapsable sections
    • [path to tyfy]/dist/script/tabs.js for responsive tabs
    • [path to tyfy]/dist/script/tables.js for responsive tables

If you want to only use parts of Tyfy, you can review the tyfy.scss file or the modules documentation and use add the specific stylesheets that are useful for your project.
