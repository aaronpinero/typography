Mixins

Tyfy includes several SASS mixins that facilitate the use of the framework and provide commonly used styles and effects.

@mixin button
Arguments
$color_default
Default button color; must be rgb, named color, or hexidecimal
Default value is $tyfy-color-link-default
$color_active
Hovered and active button color; must be rgb, named color, or hexidecimal
Default value is $tyfy-color-link-active
$border_radius
Border radius of the button; all unit values allowed
Default value is tygrid(0.25)
Purpose
Display a link or other tag as a button. The text color of the button is the default background color. To apply the button class to a link, use the [href] attribute selector.
Example
tag.class {
  @include button;
}
a.class[href] {
  @include button(red,blue,0px);
}
@mixin button-outline
Arguments
$color_default
Default button color; must be rgb, named color, or hexidecimal
Default value is $tyfy-color-link-default
$color_active
Hovered and active button color; must be rgb, named color, or hexidecimal
Default value is $tyfy-color-link-active
$border_radius
Border radius of the button; all unit values allowed
Default value is tygrid(0.25)
Purpose
Display a link or other tag as an outlined button. To apply the button class to a link, use the [href] attribute selector.
Example
tag.class {
  @include button-outline;
}
a.class[href] {
  @include button-outline(red,blue,0px);
}
@mixin clearfix
Arguments
None
Purpose
Clear floats within an element; code from HTML5 Boilerplate.
Example
tag.class {
  @include clearfix;
}
@mixin fade-in
Arguments
$duration
Duration in seconds or milliseconds
Default value is 1000ms
Purpose
Apply a (non-looping) fade-in animation to an element with the given animation duration.
Example
tag.class {
  @include fade-in(2s);
}
@mixin set-sibling-margins
Arguments
$context
Allowed values are all, headings, or lists
Default value is all
$measure
Allowed values are rem-unit lengths
Default value is $tyfy-spacing-standard
Purpose
Set the size of adjacent sibling margins in a given context. The all context sets the value for all tags; the headings context sets the value only for headings; the lists context sets the value only inside of ordered, unordered, and definition lists. For more information, see the Margins documentation.
Example
tag.class {
  @include set-sibling-margins(all,tygrid(1));
}
@mixin sibling-margins
Arguments
$state
Allowed values are on or off
Default value is on
Purpose
Turn adjacent sibling margins on or off within a specific context. If the state is on, the margins are set to the Tyfy default values. To override with specific values, use the set-sibling-margins mixin.
Example
tag.class {
  @include sibling-margins(off);
}
@mixin visuallyhidden ($type:true)
Customized from HTML5 Boilerplate; the element is not visually rendered, but the data is still present for screen readers and search indices.
