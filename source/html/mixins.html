Mixins

Tyfy includes several SASS mixins that facilitate the use of the framework and provide commonly used styles and effects.

@mixin button ($color_default:$tyfy-color-link-default,$color_active:$tyfy-color-link-active,$border_radius:tygrid(0.25))
A mixin for displaying a link or other tag as a button.
@mixin button-outline ($color_default:$tyfy-color-link-default,$color_active:$tyfy-color-link-active,$border_radius:tygrid(0.25))
A mixin for displaying a link or other tag as an outlined button.
@mixin clearfix
From HTML5 Boilerplate; clear floats within an element.
@mixin fade-in ($duration:1000ms)
Apply a fade-in animation to an element with the given animation duration.
@mixin set-sibling-margins ($context:all,$measure:$tyfy-spacing-standard)
Set the value of the margin between adjacent siblings; context values are "all", "headings", or "lists".
@mixin sibling-margins ($state:on)
Turn on or off the adjacent siblings margins; state values are "on" or "off".
@mixin visuallyhidden ($type:true)
Customized from HTML5 Boilerplate; the element is not visually rendered, but the data is still present for screen readers and search indices.
