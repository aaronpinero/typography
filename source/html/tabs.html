Tabs

It is occasionally useful to present content as a set of tabs. Tabs are presented horizontally with the content of the selected tab appearing directly below the tabs.

Unfortunately, tabs are not a good choice when the browser window is specially small or narrow, as on phones. Tyfy tabs are responsive. At narrow screen sizes, the tabs convert to collapsable headings.

Tyfy tabs accommodate linking to anchors within tabs. Each tab label is also given a unique id attribute, allowing for links that open a specific tab.

HTML Structure

All tab labels and tab content are enclosed in a block-level element with the tabs-container class. Individual tabs are defined by indicating the label for each tab with the tab-label class. JavaScript is used to identify the contents of each tab and to generate the clickable tabs. The content of a tab is defined as all the next sibling elements following the tab-label element up to the next tab-label.

<div class="tabs-container">
  <h2 class="tab-label">Tab Label 1</h2>
  <p>[content of the tab]</p>
  <p>[content of the tab]</p>
  <h2 class="tab-label">Tab Label 2</h2>
  <p>[content of the tab]</p>
  <ul>[content of the tab]</ul>
</div>

Related Variables

$tyfy-break-tabs
Breakpoint at which tabs transition from tabs to collapsables; !default value is 40em.
$tyfy-width-border
Width of the border around tab labels and tab panels; !default value is 2px.
$tyfy-color-border
Color of the borders around tab labels and tab panels; !default value mix($tyfy-color-foreground,$tyfy-color-background,12%).
