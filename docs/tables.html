Tables

Classes for table styles are defined to allow style rules to be “layered onto” a table. The clean class is the basic, required class. Additional classes apply zebra-striping, borders, and responsive behavior. The responsive behavior is enabled using both CSS and JavaScript.

Proper application of styles and the responsive behavior depends on semantically correct structure of the tables. Tables should use the <thead> and <tbody> tabs. Table headers should have the scope attribute to indicate how the headers should be interpreted.

Using the float-left and float-right classes, a table can be positioned to the left or right of the main text. Positioning left or right is limited to larger screen sizes. At smaller screen sizes, the styled tables are displayed at full width.

Classes

<table class="clean">
Display a table using Tyfy style rules. A table with this class has width set to 100% and properly padded table cells.
<table class="horizontal-rules">
Display a table with horizontal rules separating the rows.
<table class="responsive">
Display a responsive table. At browser window widths less than the $tyfy-break-table breakpoint, the table is displayed as a list of labeled values.
<table class="striped">
Display table rows with "zebra" striping. The variable $tyfy-color-shade defines the stripe color.
<table class="vertical-rules">
Display a table with vertical rules separating the columns

Variables

$tyfy-break-table: 40em !default;
Breakpoint below which a responsive table is displayed as a list of labeled values.
$tyfy-color-border: mix($tyfy-color-text-body,$tyfy-color-background,12.5%) !default;
The color of borders for tables with the horizontal-rules or vertical-rules classes
$tyfy-color-shade: mix($tyfy-color-text-body,$tyfy-color-background,7.5%) !default;
The color of zebra stripes for tables with the striped class.
$tyfy-font-table: $tyfy-sans !default;
The font family to use for table cells.
$tyfy-width-border: 2px !default;
The width of borders for tables with the horizontal-rules or vertical-rules classes. The width of borders separating table header cells from table data cells is twice the value of $tyfy-width-border.
