Quotes

The <blockquote> tag is commonly used for both long quotations and pull quotes. The basic <blockquote> style is intended for long quotations. A pullquote class can be applied to the <blockquote> tag for fancier quotes. Pull quotes can also be positioned to the left or right of the text by applying the float-left and float-right classes. Positioning left or right is limited to larger screen sizes. At smaller screen sizes, the pull quotes are displayed at full width.

The pull quote style is inspired by the design of the Core to Commencement Campaign website that was designed by Barrel. This version omits the need for images or svg, instead using pseudo-elements and a icon webfont.
