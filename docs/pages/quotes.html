Quotes

The <blockquote> tag is commonly used for both long quotations and pull quotes. Tyfy provides classes that allow the basic <blockquote> to also be used for fancier pull quotes.

Classes

.pullquote
Applied to a <blockquote> to render as a pull quote.
.pullquote-roof-fixed
Added to the pullquote class to display a decorative, fixed-width border element above the pullquote; the decorative border is the $tyfy-quote-roof icon.
.pullquote-roof-full
Added to the pullquote class to display a decorative, relative-width border element above the pullquote; the decorative border is a background svg based on the the $tyfy-quote-roof icon.
.float-left
Added to the pullquote class to position the pull quote float-left. This degrades to a regular pull quote at screen widths smaller than $tyfy-break-floats.
.float-right
Added to the pullquote class to position the pull quote float-right. This degrades to a regular pull quote at screen widths smaller than $tyfy-break-floats.

Related Variables

$tyfy-font-size-pullquote
Font size for <blockquote> with the pullquote class.
$tycolor-border
Color of the decorative border on pullquotes using the pullquote-roof-fixed or pullquote-roof-full class.
$tyfy-indent-standard
Horizontal margin for <blockquote> elements. This margin reduces to $tyfy-indent-standard at screen widths below below $tyfy-break-pullquote-large.
$tyfy-break-floats
Screen width breakpoint above which the float classes take effect.
$tyfy-break-floats
Screen width breakpoint above which quotes have increased margin.
