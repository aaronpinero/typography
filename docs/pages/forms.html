Forms

Tyfy provides basic styles for form elements. The styles are intended for simple, well structured forms where each form element consists of a label-input pair. Standard element spacing has been adjusted so labels are visually associated with the corresponding input. There are helper classes for select and label elements that provide some presentation options.

Classes

<tag class="tyfy-select">
Display a select box with custom styles to match other input elements; the class can be applied to the select element or to a containing element.
<label class="label-option">
Display a label adjacent to or encapsulating a radio button or checkbox input.
<label class="label-inline">
Display a label as an inline-block instead of block element.

Variables

$tyfy-width-border: 2px !default;
Width of the border around inputs, styled select boxes, and fieldsets.
$tyfy-color-border: mix($tyfy-color-text-body,$tyfy-color-background,12.5%) !default;
Color of the borders around inputs, styled select boxes, and fieldsets.
$tyfy-color-link-active: #cc3380 !default;
Color of the border around inputs and styled select boxes that have the input focus.

Examples

Text Inputs

<form>
  <div>
    <label>Name</label>
    <input type="text" name="name">
  </div>
  <div>
    <label>Age</label>
    <input type="number" name="age">
  </div>
</form>

Select Boxes

<form>
  <div>
    <label>Favorite Color</label>
    <select name="color">
      <option value="red">Red</option>
      <option value="green">Green</option>
      <option value="blue">Blue</option>
    </select>
  </div>
  <div>
    <label>Favorite Animal</label>
    <select name="color" class="tyfy-select">
      <option value="lion">Lion</option>
      <option value="zebra">Zebra</option>
      <option value="elephant">Elephant</option>
    </select>
  </div>
</form>

Checkboxes and Radios

<form>
  <div>
    <label>Tyfy is awesome.</label>
    <div>
      <label class="label-option label-inline"><input type="radio" name="tyfy" value="true"> True</label>
      <label class="label-option label-inline"><input type="radio" name="tyfy" value="false"> False</label>
    </div>
  </div>
  <div>
    <label>Check all that apply:</label>
    <ul class="list-nobullet sibling-margins-off">
      <li><label class="label-option"><input type="checkbox" name="apply" value="awesome"> Awesome</label></li>
      <li><label class="label-option"><input type="checkbox" name="apply" value="useful"> Useful</label></li>
      <li><label class="label-option"><input type="checkbox" name="apply" value="designed"> Well-designed</label></li>
    </ul>
  </div>
</form>

Fieldset

<form>
  <fieldset>
    <legend>Payment Information</legend>
    <div class="grid-2">
      <div>
        <label>Type of Card</label>
        <select name="card">
          <option value="visa">Visa</option>
          <option value="mastercard">Mastercard</option>
          <option value="amex">American Express</option>
        </select>
      </div>
      <div>
        <label>Card Number</label>
        <input type="text" name="card_number">
      </div>
    </div>
  </fieldset>
</form>

Button

Form buttons use the same styles as the button class. Both are based on the button mixin.

<form>
  <input type="submit" name="submit" value="Submit Form">
</form>
