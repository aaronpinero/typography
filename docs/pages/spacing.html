Spacing

The spacing module turns off the default top and bottom margins for all HTML elements, replacing those styles with Tyfys adjacent sibling margin rules. The adjacent sibling margin rules are applied if the value of the $tyfy-margins-on setting is true.

Horizontal Rule

Tyfy treats the horizontal rule tag (<hr>) as a special case. In the spacing module, this tag’s top and bottom margin are set to the value of the $tyfy-spacing-large variable.

Classes

The following classes are defined by the spacing module. They can be used to enable or disable the adjacent sibling margins in specific contexts. Please note: these classes may not behave as expected if nested. So, it is not recommended to use the .sibling-margins-on class on a descendant of an element that is using the .sibling-margins-off, or vice versa. This is because of how CSS interprets the wildcard selector.

.sibling-margins-on
Apply the default adjacent sibling margin rules to descendants of the element with this class
.sibling-margins-off
Remove the default adjacent sibling margin rules from descendants of the element with this class
