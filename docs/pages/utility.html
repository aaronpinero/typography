Styles

Tyfy includes style customizations for block and inline HTML elements used in typographic design. In addition, custom classes are defined for additional typographic enhancement.

Special Case: Horizontal Rule

The horizontal rule (<hr>) has unique margin styles. While all other elements in the framework inherit by the adjacent sibling margin styles, the horizontal rule is given both top and bottom margin. This exception is justified by the purpose of the horizontal rule, which is to act as a separator between other elements.

The top and bottom margins are equal to the value of the $tyspacing-lg variable. The margins are applied in two cases:

  • when the value of the $tymargins-on variable is true; or
  • when a parent element has the .tymargins-on class.
