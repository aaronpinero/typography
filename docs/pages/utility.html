Styles

Tyfy includes style customizations for other block and inline HTML elements. There are also classes for styles that do not have an HTML equivalent.

Sample Tag.Class Description
Bold text b
strong		 Bold text
Italic text em
i		 Italic text
Code code Text describing code computer code
Code, no background code.code-nobg The code tag with no background color 
Preformatted
 pre Preformatted text
Small small Small text; makes text one scale step smaller
hr Horizontal rule; the thickness is $tyfy-width-border; the color is $tycolor-border
Light .light Lighter text; the color is $tycolor-text-light
Small Caps span.smallcaps Small caps text
Button .button Text as a button
Outlined .button-outline Text as an outlined button
Drop cap span.dropcap A decorative drop cap
Sans-serif font .font-sans Display text using the $tyfy-sans font
Serif font .font-serif Display text using the $tyfy-serif font
Monospace font .font-mono Display text using the $tyfy-mono font
