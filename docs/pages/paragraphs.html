Paragraphs

Paragraphs inherit font size and line height from the <body>. Utility classes allow paragraphs to be used as a lede or as justified text.

Classes

.paragraphs-justified
Applied to a containing element to a make all child paragraphs justified.
.justified
Applied to a <p> tag to justify that paragraph.
.lede
Applied to a <p> tag to display the paragraph as a lede, with increased font size and line height. The line height and font size are the same as that for pull quotes.
.cols-2
Applied to a containing element to a make child content display in 2 columns using CSS columns. The style is responsive and reduce to a single column for small screen widths.
