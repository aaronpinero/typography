Collapsable Headings

Collapsable headings are a simplified version of the accordion pattern. Tyfy collapsable headings:

  • are easy to implement
  • are responsive
  • can be nested
  • allow for anchored links inside collapsed content
  • require Javascript (collapsables.js)

Content authors can employ this pattern using single CSS class applied to a heading (level 2 through 6). JavaScript encapsulates all of the content belonging to that heading in a block element, collapses the content, and binds a click event to the heading that toggles the display of the content. This makes collapsable headings easy to implement and apply in simple text editors like CKEditor.

The display of collapsable headings is responsive. At larger screen sizes, each collapsable heading and content is fully bordered with padding equal to two grid units. At small screen widths, the design changes, removing the full border and padding to maximize horizontal space.

Collapsable headings can be nested. They also accommodate linking to anchors within collapsed content. The correct collapsed area will open to show the link target.

HTML Structure

Collapsable headings are defined by adding the collaspable class to a heading (H2 through H6).

<h2 class="collapsable">Collapsable Heading</h2>
<p>[content under the heading]</p>
<ul>[content under the heading]</ul>

<h2>Regular Heading 2</h2>
<p>[content under the heading]</p>
<ul>[content under the heading]</ul>

JavaScript is used to identify the contents of belonging to the heading. The content of a collapsable section is defined as all the next sibling elements following the collaspable heading element up to the next heading of the same level. JavaScript wraps the content of the collapsable section in a <div> with the class collapsables. To indicate that JavaScript has acted on the collapsable heading, the class collapsable-processed is added to the collapsable heading. So, after JavaScript has run, the code will appear as follows:

<h2 class="collapsable collapsable-processed">Collapsable Heading</h2>
<div class="collapsables">
  <p>[content under the heading]</p>
  <ul>[content under the heading]</ul>
</div>

<h2>Regular Heading 2</h2>
<p>[content under the heading]</p>
<ul>[content under the heading]</ul>

Variables

$tyfy-break-collapsables-pad
50em !default;
Screen width below which the collapsable heading design changes to maximize horizontal space
$tyfy-width-border
2px !default;
Thickness of the borders defining collapsable headings and content
$tycolor-border
mix($tycolor-fg,$tycolor-bg,12%) !default;
Default border color
$tyfyicons-angle-down
"ea01" !default;
The character code for the icon indicating a collapsable section can be opened
$tyfyicons-angle-up
"ea08" !default;
The character code for the icon indicating a collapsable section can be closed
