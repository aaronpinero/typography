Settings

The settings module is a library of SASS variables that define the margins, colors, fonts, font sizes, breakpoints, and other characteristics.

Variable names begin with the prefix $tyfy- (or $tycolor- for color variables) to avoid name collision with other projects. Variable values are assigned using the !default declaration so they may be overridden when including the Tyfy SASS files in your project.

// grid and spacing

$tyfy-spacings
("standard":2, "large":3,) !default;
A SASS map of values indicating the number of grid units for each spacing variable value
$tyfy-spacing-standard
tygrid(map-get($tyfy-spacings,"standard")) !default;
Standard spacing between HTML elements defined as a number of grid units
$tyfy-spacing-large
tygrid(map-get($tyfy-spacings,"large")) !default;
Larger spacing between HTML elements defined as a number of grid units; used for spacing above headings
$tyfy-indents
("standard":3, "small":2,) !default;
A SASS map of values indicating the number of grid units for each indent variable value
$tyfy-indent-standard
tygrid(map-get($tyfy-indents,"standard")) !default;
Standard indent distance for lists and blockquotes defined as a number of grid units
$tyfy-indent-small
tygrid(map-get($tyfy-indents,"small")) !default;
Small indent distance for lists and blockquotes at small screen sizes
$tyfy-margins-on
true !default;
Determines if adjacent sibling margins styles should be applied on the <body> tag

// site colors

In the body module, the following colors are assigned to CSS custom properties to facilicate dynamic or contextual color changes. The custom property names are the same as the equivalent SCSS variable names.

$tycolor-bg
white !default;
Background color of the <body> element
$tycolor-fg
#111111 !default;
Color of the <body> element
$tycolor-text-light
mix($tycolor-fg,$tycolor-bg,45%) !default;
A light text color variant
$tycolor-link-default
#3380cc !default;
Default link color
$tycolor-link-active
#cc3380 !default;
Color of links that are hovered or active
$tycolor-bg-tab-active
rgba($tycolor-link-active,0.12) !default;
Background color for unselected tabs when hovered or active
$tycolor-border
mix($tycolor-fg,$tycolor-bg,12%) !default;
Default border color
$tycolor-border-link-default
rgba($tycolor-link-default,0.5) !default;
Default border color for underlined links
$tycolor-border-link-active
rgba($tycolor-link-active,0.75) !default;
Border color for underlined links when hovered or active
$tycolor-shade
rgba($tycolor-fg,0.05) !default;
A background color for shading
