Settings

Tyfy has a library of SASS variables that define the margins, colors, fonts, font sizes, breakpoints, and other characteristics. These variables are defined in the settings module.

All variables are assigned !defaults which can be overridden when including the Tyfy SASS files in your project.

All variables begin with the prefix $tyfy- to avoid name collision with other projects.

// grid and spacing

$tyfy-spacings: ("standard":2,"large":3,) !default;
A SASS map of values indicating the number of grid units for each spacing variable value
$tyfy-spacing-standard: tygrid(map-get($tyfy-spacings,"standard")) !default;
Standard spacing between HTML elements defined as a number of grid units
$tyfy-spacing-large: tygrid(map-get($tyfy-spacings,"large")) !default;
Larger spacing between HTML elements defined as a number of grid units; used for spacing above headings
$tyfy-indents: ("standard":3,"small":2,) !default;
A SASS map of values indicating the number of grid units for each indent variable value
$tyfy-indent-standard: tygrid(map-get($tyfy-indents,"standard")) !default;
Standard indent distance for lists and blockquotes defined as a number of grid units
$tyfy-indent-small: tygrid(map-get($tyfy-indents,"small")) !default;
Small indent distance for lists and blockquotes at small screen sizes
$tyfy-margins-on: true !default;
Determines if adjacent sibling margins styles should be applied on the <body> tag

// fonts

$tyfy-serif: georgia, serif !default;
Serif font selection
$tyfy-sans: helvetica, arial, sans-serif !default;
Sans-serif font selection
$tyfy-mono: monaco, courier, monospace !default;
Monospace font selection
$tyfy-symbol: tyfyicons !default;
Icon font selection

// font assignments

$tyfy-font-base:
Base font selection, used for body text
$tyfy-font-heading:
Heading font selection
$tyfy-font-table:
Table cell font selection
$tyfy-font-input:
Form input element font selection

// typographic scale

$tyfy-font-sizes:
A SASS map of values indicating the scale step for each of the font size variables
$tyfy-font-size-sm:
Small font size, used in the quotes and tables modules; defaults to one scale step down from the base font size
$tyfy-font-size-h6:
Level 6 heading font size
$tyfy-font-size-h5:
Level 5 heading font size
$tyfy-font-size-h4:
Level 4 heading font size
$tyfy-font-size-h3:
Level 3 heading font size
$tyfy-font-size-h2:
Level 2 heading font size
$tyfy-font-size-h1:
Level 1 heading font size
$tyfy-font-size-pullquote:
Pullquote font size; !default value is tyscale(2)
$tyfy-font-size-lede:
Lede font size; !default value is tyscale(2)

// measures

$tyfy-width-border:
Default thickness for borders; !default value is 2px
$tyfy-radius-border:
Default border radius for elements with rounded corners; !default value is 4px

// breakpoints

$tyfy-break-floats:
Below this width floated elements are not floated, instead shown at 100% width
$tyfy-break-pullquote-large:
Above this width the margins around pullquotes increase
$tyfy-break-cols-2:
Above this width content using the cols-2 class can be 2 columns; !default value is 50em
$tyfy-break-list-cols-2:
Above this width lists can be 2 columns
$tyfy-break-list-cols-3:
Above this width lists can be 3 columns
$tyfy-break-list-cols-4:
Above this width lists can be 4 columns
$tyfy-break-collapsables-pad:
Below this screen width, the style of collapsables changes to maximize horizontal space; !default value is 50em
$tyfy-break-tabs:
Below this width responsive tabs convert to collapsable sections
$tyfy-break-table:
Below this width responsive tables are converted to labeled lists

// transitions

$tyfy-transition-duration-hover:
Duration in seconds for CSS transitions where colors are changing
$tyfy-transition-duration-move:
Duration in seconds for CSS transitions where measures are changing

$tyfy-color-link-active:
Color of links that are hovered or active
$tyfy-color-border:
