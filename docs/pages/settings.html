Settings

Tyfy has a library of SASS variables that define the margins, colors, fonts, font sizes, breakpoints, and other characteristics. These variables are defined in the settings module.

All variables begin with the prefix $tyfy- to avoid name collision with other projects. Variable values are assigned using the !default declaration so they may be overridden when including the Tyfy SASS files in your project.

// grid and spacing

$tyfy-spacings
("standard":2, "large":3,) !default;
A SASS map of values indicating the number of grid units for each spacing variable value
$tyfy-spacing-standard
tygrid(map-get($tyfy-spacings,"standard")) !default;
Standard spacing between HTML elements defined as a number of grid units
$tyfy-spacing-large
tygrid(map-get($tyfy-spacings,"large")) !default;
Larger spacing between HTML elements defined as a number of grid units; used for spacing above headings
$tyfy-indents
("standard":3, "small":2,) !default;
A SASS map of values indicating the number of grid units for each indent variable value
$tyfy-indent-standard
tygrid(map-get($tyfy-indents,"standard")) !default;
Standard indent distance for lists and blockquotes defined as a number of grid units
$tyfy-indent-small
tygrid(map-get($tyfy-indents,"small")) !default;
Small indent distance for lists and blockquotes at small screen sizes
$tyfy-margins-on
true !default;
Determines if adjacent sibling margins styles should be applied on the <body> tag

// measures

$tyfy-width-border
2px !default;
Default border thickness for many HTML elements including rules, table cells, form elements, and link underlines.
$tyfy-radius-border
4px !default;
Default border radius for elements with rounded corners including buttons, form elements, and tabs

// breakpoints

$tyfy-break-floats
40em !default;
Minimum screen width for floated quotes and tables
$tyfy-break-pullquote-large
40em !default;
Screen width above which <blockquote> indent increases from small to standard
$tyfy-break-cols-2
50em !default;
Minimum screen width at which the cols-2 class is effective
$tyfy-break-list-cols-2
35em !default;
Minimum screen width at which lists can be displayed in 2 columns
$tyfy-break-list-cols-3: 55em !default;
Minimum screen width at which lists can be displayed in 3 columns
$tyfy-break-list-cols-4: 75em !default;
Minimum screen width at which lists can be displayed in 4 columns
$tyfy-break-collapsables-pad: 50em !default;
Screen width below which the style of collapsables changes to maximize horizontal space
$tyfy-break-tabs: 40em !default;
Screen width below which the responsive tabs convert to collapsable sections
$tyfy-break-table: 40em !default;
Screen width below which responsive tables are converted to labeled lists

// transitions

$tyfy-transition-duration-hover: 0.25s !default;
Duration in seconds for CSS transitions related to hover events
$tyfy-transition-duration-move: 0.5s !default;
Duration in seconds for CSS transitions where measures (width, height, position) are changing

// site colors

$tyfy-color-background: white !default;
Background color of the <body> element
$tyfy-color-foreground: #111111 !default;
Color of the <body> element
$tyfy-color-text-light: mix($tyfy-color-foreground,$tyfy-color-background,45%) !default;
A light text color variant, defined as a function of the foreground and background colors
$tyfy-color-link-default: #3380cc !default;
Default link color
$tyfy-color-link-active: #cc3380 !default;
Color of links that are hovered or active
$tyfy-color-border: mix($tyfy-color-foreground,$tyfy-color-background,12%) !default;
Default border color for tables, pullquotes, collapsable sections, etc. defined as a function of the foreground and background colors
$tyfy-color-shade: rgba($tyfy-color-foreground,0.05) !default;
A subtle background color used in zebra-stripe tables and other places where shading is desireable in the design; defined as a function of the foreground color
