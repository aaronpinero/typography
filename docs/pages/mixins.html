Mixins

Tyfy mixins make it easier to use the some of the framework’s styles and effects..

@mixin button

Use this mixin to display a link or other tag as a button. The button’s text color will be black or white – whichever provides the best contrast with the button’s background color. To apply the mixin to a link, use the [href] attribute selector. See the links and utility documetation for more information.

Arguments

$color_default
Default button color
Default value is $tyfy-color-link-default
$color_active
Hovered and active button color
Default value is $tyfy-color-link-active
$border_radius
Border radius of the button
Default value is $tyfy-radius-border

Examples

tag.button {
  @include button;
}
a.button[href] {
  @include button(red,blue,0px);
}

@mixin button-outline

Use this mixin to display a link or other tag as an outlined button. In the default state, the button background color is transparent and the button text color is the default color. In the active state, the button background color is the active color and the text color is either black or white – whichever provides the best contrast with the background color. To apply the mixin to a link, use the [href] attribute selector. See the links and utility documetation for more information.

Arguments

$color_default
Default button color; must be rgb, named color, or hexidecimal
Default value is $tyfy-color-link-default
$color_active
Hovered and active button color; must be rgb, named color, or hexidecimal
Default value is $tyfy-color-link-active
$border_radius
Border radius of the button; all unit values allowed
Default value is $tyfy-radius-border

Examples

tag.button-outline {
  @include button-outline;
}
a.button-outline[href] {
  @include button-outline(red,blue,0px);
}

@mixin clearfix

Clear floats within an element. This mixin does not accept any arguments. Code comes from HTML5 Boilerplate.

Example

tag.class {
  @include clearfix;
}

@mixin fade-in

Apply a non-looping fade in animation to an element with the given animation duration.

Arguments

$duration
Duration in seconds or milliseconds
Default value is 1000ms

Example

tag.class {
  @include fade-in(2s);
}

@mixin visuallyhidden

Visually hide an element, but allow screen readers and search engines to read the element’s content. The default true argument hides the element. The false argument is used to override the default styles. Code is adapted from HTML5 Boilerplate

Arguments

$state
Allowed values are true or false
Default value is true

Example

tag.hide-me {
  @include visuallyhidden;
  &.show-me {
    @include visuallyhidden(false);
  }
}

@mixin set-sibling-margins ($context:all, $measure:$tyfy-spacing-standard)
A mixin for setting the value of the vertical margin between adjacent siblings. The all context employs the lobotomized owl selector for margins between all elements. The headings context applies margin between a heading and a preceding non-heading element. The lists context applies margin between child element of definition, ordered, and unordered lists.
@mixin sibling-margins ($state:on)
A mixin for turning the vertical margin between adjacent siblings on an off. It depends on the set-sibling-margins mixin.
