Link Styles

Links must have a style that is different from body text, but anchors should not. Because anchors and links share the <a> tag, it is necessary to progressively define styles for this tag. Here, the default appearance of the <a> tag is the same as for body text. Using an attribute selector, those elements that have an href attribute have link styles applied.

a {
  [styles]
  &[href] {
    [link styles]
    &:hover,
    &:active {
      [hovered and active link styles]
    }
  }
}

For underlined links, I have used border-bottom to apply an underline instead of text-decoration. This way, the underline can be styled and transitions can be employed. The framework includes SASS variables to globally define transition speeds.

Classes

.button
Applied to an <a> tag to make a link appear as a button.
.button-outline
Applied to an <a> tag to make a link appear as an outlined button.
.links-underlined
Applied to a containing element to add underlines to descendant links; by default, links do not have any underline.
.links-notunderlined
Applied to a containing element to remove underlines to descendant links; intended to be used inside an element that has the links-underlined class.

Variables

$tycolor-fg
#111111 !default;
Color of anchor links (same as body text color)
$tyfy-transition-duration-hover
0.25s !default;
Duration for transition of border, background color, and text color on hovered links.
$tycolor-link-default
#3380cc !default;
Default link color
$tycolor-link-active
#cc3380 !default;
Hovered and active link color
$tycolor-border-link-default
rgba($tycolor-link-default,0.5) !default;
Default link underline color
$tycolor-border-link-active
rgba($tycolor-link-active,0.75) !default;
Hovered and active link underline color
$tyfy-width-border
2px !default;
Width (thickness) of the link underline
