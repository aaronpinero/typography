Paragraph Styles

Paragraph styles are applied to the <p> tag and heading levels 1 through 6. SASS variables define the modular scale of font sizes and the vertical grit unit. Font sizes are available as both SASS variables and CSS classes.

Additional variables define available text colors and font families. For this demonstration, I have selected fonts that are common in my work as a Web Developer at Columbia University. FontAwesome 5, for which I purchased a pro license through the Kickstarter campaign, is the selected icon font. I highly recommend supporting FontAwesome.

My preference for the use of headings is to have each heading positioned closer to the text below than the text above. Default margins for paragraphs elements are zero, and combination selectors are used to define styles for paragraph and headings in context of other paragraph elements.

In my design work, I am often asked to balance the request for more contrast between heading levels with a limit on the size of the <h1> text. I have used style, weight, and text transforms in an effort to create contrast without relying only on font size.
