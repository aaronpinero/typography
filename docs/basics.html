Tyfy

Basics

Tyfy has a library of SASS variables that define the base grid, colors, text sizes, fonts, breakpoints, and other characteristics. My goal with this project is to have as few variables as possible. This results in some key variables – such as the grid unit, brand color and font size scalar – that can quickly change the entire character of the project to which Tyfy is applied.

SASS Variables

Colors

$color-brand
The primary color, used for links and other elements
$color-text-body
The default text color for paragraphs and lists
$color-text-head
The default text color for headings
$color-text-light
A secondary text color used for labels and other places where contrast from the default color is useful
$color-text-pullquote
Color for pullquote text
$color-link-default
Default color for linked text and buttons
$color-link-active
The hover and active color for links and buttons

Reverse Colors

$color-text-body-reverse
The reverse text color, defined as invert($color-text-body)
$color-text-head-reverse
The reverse heading color, defined as invert($color-text-head)
$color-text-light-reverse
The reverse secondary text color, defined as invert($color-text-light)
$color-text-pullquote-reverse
The reverse pull quote text color, defined as saturate(lighten($color-text-pullquote,30%),30%)
$color-link-default-reverse
The reverse default link color, defined as saturate(lighten($color-link-default,30%),30%)
$color-link-active-reverse
The reverse hover and active link color, defined as complement($color-link-default-reverse)

Breakpoints

$break-floats
Screen width above which position classes (float left and right) are applied
$break-pullquote-large
Screen width above which non-floated pull quotes receive increased padding
$break-list-cols-2
Screen width above which two column list styles are active
$break-list-cols-3
Screen width above which three column list styles are active

Measures

$grid-unit
Basic unit of the vertical grid and horizontal measures (such as element padding); represents half the default body text line height
$font-size-scalar
Value representing the ratio between type sizes in the Tyfy typographic scale; the default value is 1.2 (a minor third)
$font-size-p
The base size of the typographic scale; the default body text size
$font-size-sm
The size of small text, defined as $font-size-p / $font-size-scalar
$font-size-h6
The size of a level 6 heading, defined as $font-size-p / $font-size-scalar
$font-size-h5
The size of a level 5 heading, defined as $font-size-p
$font-size-h4
The size of a level 4 heading, defined as $font-size-h5 * $font-size-scalar
$font-size-h3
The size of a level 3 heading, defined as $font-size-h4 * $font-size-scalar
$font-size-h2
The size of a level 2 heading, defined as $font-size-h3 * $font-size-scalar
$font-size-h1
The size of a level 1 heading, defined as $font-size-h2 * (2 * $font-size-scalar)
$font-size-pullquote
The size of pull quote text, defined as $font-size-p * $font-size-scalar
