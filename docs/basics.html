Basics

Tyfy has a library of SASS variables that define the base grid, colors, text sizes, fonts, breakpoints, and other characteristics. My goal with this project is to have as few variables as possible. This results in some key variables – such as the grid unit, brand color and font size scalar – that can quickly change the entire character of the project to which Tyfy is applied.

Vertical rhythm is maintained by using the base grid unit to define line heights. Unlike other frameworks like Bootstrap, which set a relative line height for all elements, Tyfy line heights are multiples of the base grid unit. Calculation of grid-based line heights is aided by a custom SASS function.

Fonts

There are four font selections for Tyfy. While there are many way in which typefaces could be selected and organized I have selected the way which works for most of the projects I manage. My practice is to limit the number of typefaces involved in a project.

$tyfy-serif
The serif font selection for the project
$tyfy-sans
The sans-serif font selection for the project
$tyfy-mono
The monospace font selection for the project
$tyfy-symbol
The icon font for the project; for my projects, this typeface is almost always FontAwesome
$tyfy-font-base
The typeface assignment for the <body>
$tyfy-font-heading
The typeface assignment for headings 1 to 6
$tyfy-font-pullquote
The typeface assignment for pull quotes
$tyfy-font-table
The typeface assignment for table cells
$tyfy-font-size-scalar
The value used as a multiplier for creating the typographic scale of font sizes

Font sizes are calculated using a scalar value, creating a typographic scale of font sizes that are proportionally related. The typographic scale can be easily adjusted by changing the scalar value and/or the base font size.

$tyfy-font-size-base
The font size of the <body>; needs to be specified in rem units
$tyfy-font-size-sm
The font size for small type, calculated as $tyfy-font-size-base / $tyfy-font-size-scalar
$tyfy-font-size-h6
The font size for the level 6 heading, set to $tyfy-font-size-base
$tyfy-font-size-h5
The font size for the level 5 heading, calculated as $tyfy-font-size-h6 * $tyfy-font-size-scalar
$tyfy-font-size-h4
The font size for the level 4 heading, calculated as $tyfy-font-size-h5 * $tyfy-font-size-scalar
$tyfy-font-size-h3
The font size for the level 3 heading, calculated as $tyfy-font-size-h4 * $tyfy-font-size-scalar
$tyfy-font-size-h2
The font size for the level 2 heading, calculated as $tyfy-font-size-h3 * $tyfy-font-size-scalar
$tyfy-font-size-h1
The font size for the level 1 heading, calculated as $tyfy-font-size-h2 * $tyfy-font-size-scalar * $tyfy-font-size-scalar
$tyfy-font-size-pullquote
The font size for the level 6 heading, set to $tyfy-font-size-h4

Colors

$tyfy-color-link
The primary color, used for links and other elements
$tyfy-color-text-body
The default text color for paragraphs and lists
$tyfy-color-text-head
The default text color for headings
$tyfy-color-text-light
A secondary text color used for labels and other places where contrast from the default color is useful
$tyfy-color-text-pullquote
Color for pullquote text
$tyfy-color-link-default
Default color for linked text and buttons
$tyfy-color-link-active
The hover and active color for links and buttons
$tyfy-color-border
Color for table cell borders
$tyfy-color-table-stripe
When a table has the zebra-stripe style, the row colors alternate between transparent and this stripe color

Reverse colors are defined for use with the reverse class. This class applies the correct color styles in a context where the background color or brightness is inverted.

$tyfy-color-text-body-reverse
The reverse text color, defined as invert($tyfy-color-text-body)
$tyfy-color-text-head-reverse
The reverse heading color, defined as invert($tyfy-color-text-head)
$tyfy-color-text-light-reverse
The reverse secondary text color, defined as invert($tyfy-color-text-light)
$tyfy-color-text-pullquote-reverse
The reverse pull quote text color, defined as saturate(lighten($tyfy-color-text-pullquote,30%),30%)
$tyfy-color-link-default-reverse
The reverse default link color, defined as saturate(lighten($tyfy-color-link-default,30%),30%)
$tyfy-color-link-active-reverse
The reverse hover and active link color, defined as complement($tyfy-color-link-default-reverse)

Breakpoints

$tyfy-break-floats
Screen width above which position classes (float left and right) are active
$tyfy-break-pullquote-large
Screen width above which non-floated pull quotes receive increased padding
$tyfy-break-list-cols-2
Screen width above which two column list styles are active
$tyfy-break-list-cols-3
Screen width above which three column list styles are active
$tyfy-break-tabs
Screen width below which responsive tabs switch from tabs to collapsable sections
$tyfy-break-table
Screen width below which responsive tables convert default table structure to cell list

Grid

The grid unit value is used to calculate line heights, margins and paddings throughout Tyfy. Line height calculation is aided by the leading() function. The leading function takes two arguments: a font size and a line height option. The line height option can be:

  • base: a line height equal to some multiple of the grid unit that is greater than the font size.
  • tall: a line height equal to some multiple of the grid unit that is much greater than the font size.
  • heading: a line height equal to some multiple of the grid unit greater than or equal to the font size.
  • short: a line height equal to some multiple of the grid unit that is more or less equal to the font size.

The variations in line height style are calculated with the aid of an adjustment value. Changing the adjustment values will change the relationship between the font size and the resulting line-height.

The leading function can be used to calculate other style property values in cases where some measure (height, width, top, etc.) needs to equal a specific line height.

$tyfy-grid-unit
Basic unit of the vertical grid and horizontal measures (such as element padding); represents half the base line height
$tyfy-leading-adjustment-tall
A value used to modify line height calculations to result in larger values; set to 0.8
$tyfy-leading-adjustment-base
A value used to modify line height calculations, making sure the line height will always be more than the font size; set to 0.9
$tyfy-leading-adjustment-heading
A value used to modify line height calculations to allow slightly shorter line heights for headings; set to 1
$tyfy-leading-adjustment-short
A value used to modify line height calculations to allow line heights that are slightly shorter than the font size; set to 1.2
