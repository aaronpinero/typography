Optional Modules

The optional modules provide styles for HTML element and CSS classes for specific typographic styles. All optional modules are included in the default distribution.

Because not every project will need all of these styles, the modules are considered optional. When customizing Tyfy for your project, you can include the optional modules that best fit your needs.

  • Paragraphs
  • Headings
  • Links
  • Lists
  • Typographic Utilities
  • Tables
  • Collapsable Sections
  • Responsive Tabs
  • Forms
  • Effects
