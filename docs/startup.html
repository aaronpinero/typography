Get Started

Overview

Tyfy is intended as a companion to your web project. It is a collection of SCSS stylesheets and jQuery-based JavaScript that provide a baseline grid, modular scale, and intelligent margins. The styles can be customized by overriding the default SASS variable values.

I have tried to make Tyfy as modular as possible. You can use all of it (as CSS or SCSS) or pick the modules that are right for your project. JavaScript is only required for certain presentation styles, and the JavaScript files are modular; each script is required only when using the associated module.

Setup

  1. Clone the github project or install using npm install tyfy
  2. Add Tyfy to your project:
    • As s SCSS file: @import "[path to tyfy]/source/scss/tyfy.scss"
    • As a CSS file: @import "[path to tyfy]/dist/style/tyfy.css"
  3. Add jQuery (1.12.4 or above) to your project
  4. Add Tyfy JavaScript to your project in the contexts where it is appropriate.
    • [path to tyfy]/dist/script/collapsables.js for collapsable sections
    • [path to tyfy]/dist/script/tabs.js for responsive tabs
    • [path to tyfy]/dist/script/tables.js for responsive tables

If you want to only use parts of Tyfy, you can review the tyfy.scss file or the modules documentation and use add the specific stylesheets that are useful for your project.
