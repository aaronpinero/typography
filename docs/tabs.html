Tabs

It is occasionally useful to present content as a set of tabs. The available tabs are presented horizontally with the content of the selected tab appearing directly below the tabs.

Unfortunately, tabs are not a good choice when the browser window is specially small or narrow, as on phones. Tyfy tabs are responsive. At narrow screen sizes, the tabs convert to collapsable headings.

Like the collapsable headings, tabs are defined by indicating the label for each tab with the tab-label class. All tab labels and tab content are enclosed in a block-level element with the tabs-container class. JavaScript is used to identify the contents of each tab panel and to generate the clickable tabs.

This implementation also accounts for anchor links within tabs. Anchor links can also be used to link directly to a tab.
