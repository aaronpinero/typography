Browser Compatibility

Tyfy depends on the support of the following CSS standards. Some styles may not work properly if your target web browser lacks the necessary support. You can assess support for each standard at caniuse.com.

  • CSS Media Queries
  • calc() as CSS unit value
  • pseudo-elements
  • CSS Hyphenation
  • CSS3 Multiple column layout
  • CSS3 Transitions
  • CSS Animation
  • CSS Flexbox
  • CSS Appearance
  • CSS3 Selectors
  • CSS Custom Properties
