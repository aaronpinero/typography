List Styles

Ordered, unordered, and definition lists inherit styles from the essential/body and essential/spacing modules. In addition to the default list types, Tyfy offers several display options for ordered and unordered lists.

Classes

.list-icons
Applied to an unordered list to replace the default bullets with icons. A second class is required to identify the icon to be used. Tyfy includes the list-icons-angle class which uses the small angle right icon. A project can define additional icon classes based on list-icons-angle.
.list-nobullet
Applied to an unordered list to remove the bullets and left-hand padding. When using this class, you can add the list-indented class to more clearly identify individual list items. The list-indented class indents all lines after the first line in each list item.
.list-cols-2, .list-cols-3
Applied to ordered and unordered lists to display items in columns. These classes use CSS columns; columns are read top to bottom and then left to right. For unordered lists, these classes can be used with list-icons or list-nobullet.
.list-grid-2, .list-grid-3, .list-grid-4
Applied to ordered and unordered lists to display items in rows. These classes use CSS flexbox; rows are read left to right and then top to bottom. For unordered lists, these classes can be used with list-icons or list-nobullet.

Related Variables

$tyfy-font-size-base
Lists inherit this font size from the <body>.
$tyfy-color-foreground
Lists inherit this font color from the <body>.
$tyfy-spacing-standard
$tyfy-spacing-standard is the margin between lists and other elements. Inside lists, $tyfy-spacing-standard/2 is the spacing between list items.
$tyfy-indent-standard
The amount of indentation for list items, definition list definitions, and indented text using the indented class; applied using padding-left.
$tyfy-font-base
Lists inherit this font family from the <body>.
