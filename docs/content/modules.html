Modules

The tyfy.scss file is an inventory of the modules that make up the framework. The required, essential, and optional modules are are @imported in the following order. The extras modules are not, by default, part of Tyfy but can be added if desired.

Required

These modules are required to use Tyfy. They define the framework's variables, functions, and mixins.

@import "required/core"
Defines and provides default values for the core Tyfy variables; these variables are used in the calculation of many of the framework settings and are required by the functions module
@import "required/functions"
Defines functions that facilitate use of the grid and modular typographic scale
@import "required/settings"
Defines and provides default values for the framework’s colors, fonts, breakpoints, etc.
@import "required/mixins"
Defines SASS mixins that facilitate the application of the framework’s styles and effects

Essential

These modules are not hard requirements for Tyfy but are highly recommended.

@import "essential/normalize"
Selected styles from normalize.css
@import "essential/body"
Apply the base colors, font styles, and line height
@import "essential/spacing"
Apply 'smart' vertical margins between all adjacent sibling HTML elements; also contains utility classes for turning on and off the adjacent sibling margins

Optional

These optional modules provide specific typographic or presentation styles.

@import "optional/paragraphs"
Styles and utility classes for paragraphs
@import "optional/headings"
Styles and utility classes for headings
@import "optional/links"
Styles and utility classes for links
@import "optional/lists"
Styles and utility classes for lists
@import "optional/inline"
Styles and utility classes for inline-level elements
@import "optional/quotes"
Styles and utility classes for blockquotes and pullquotes
@import "optional/tables"
Styles and utility classes for tables; responsive tables requires tables.js
@import "optional/reverse"
A utility class for reversed colors
@import "optional/collapsables"
A presentation style for collapsable sections; requires collapsable.js
@import "optional/tabs"
A presentation style for responsive tabs; requires tabs.js
@import "optional/forms" beta
Styles and utility classes for form elements
@import "optional/effects" experimental
A SASS map that generates classes for CSS animations

Extras

These modules are not part of the default Tyfy stylesheet but can be added if desired. They provide additional functionality that might not be needed in every case. See the Extras documentation for more information.

@import "extras/icons"
Defines the font face and classes for a custom icon webfont
@import "extras/drupal"
Tyfy-based style overrides for user interface elements in Drupal
@import "extras/responsive_text"
Text size adjustments for small and large screens
