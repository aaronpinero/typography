Core

Tyfy has a library of SASS variables that define the baseline grid, colors, text sizes, fonts, breakpoints, and other characteristics. The core variables are those that affect the baseline grid and modular scale. The value of these variables can have a significant impact on the character of your typographic design.

Core variables are stored in the required/core module. This module is the first one imported to the framework. Core variable values are required by the required/functions module to calculate values for variables in the required/settings module.

All variable names begin with the prefix $tyfy- to avoid name collision with other projects.

Core Variables

$tyfy-grid-unit: 0.875rem !default;
Base grid unit in rem units; I recommend this value be half the desired line height
$tyfy-leading-adjustment-tall: 0.8 !default;
Scalar value used by the tylead function to calculate line height; values < 1 result in taller leading
$tyfy-leading-adjustment-base: 0.9 !default;
Scalar value used by the tylead function to calculate line height; values < 1 result in taller leading
$tyfy-leading-adjustment-heading: 1 !default;
Scalar value used by the tylead function to calculate line height; values >= 1 result in smaller leading
$tyfy-leading-adjustment-short: 1.2 !default;
Scalar value used by the tylead function to calculate line height; values > 1 result in smaller leading
$tyfy-font-size-scalar: 1.2 !default;
Scalar value that defines the proportion between text sizes on the typographic scale
$tyfy-font-size-base: 1rem !default;
Font size in rem units applied to the <body> tag
