Headings

For heading levels 1 through 6, SASS variables define the modular scale of font sizes. Heading font sizes also available as SASS variables and CSS classes.

My preference for the use of headings is to have each heading positioned closer to the text below than the text above. Adjacent sibling selectors are used to define styles for headings in context of other elements. The margin above a heading is larger than below.

In my design work, I am often asked to balance the request for more contrast between heading levels with a limit on the size of the <h1> text. Style, weight, and text transforms create contrast without relying only on font size.

Classes

.font-size-h[x]
Apply the font size of heading level [x] to an HTML element. This applies only the font size and not line height or other styles.

Related Variables

$tyfy-font-size-h[x]
Variable that defines the font size for heading level [x].
$tyfy-spacing-large
Extra spacing between HTML elements; used for spacing above headings; default value is grid(3).
$tyfy-font-heading
Heading font selection; default value is $tyfy-serif.
$tyfy-color-text-head
Default heading text color; default value is 95% of the foreground color mixed with 5% of the background color.
$tyfy-leading-adjustment-heading
Unitless value used in by the leading function to calculate heading leading; values less than 1 result in larger leading; default value is 1.
