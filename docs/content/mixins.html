Mixins

Tyfy mixins make it easier to use the some of the framework’s styles and effects.

@mixin button

Use this mixin to display an HTML element as a button. The button’s text color will be black or white – whichever provides the best contrast with the button’s background color. To apply the mixin to a link, use the [href] attribute selector. See the links and utility documetation for more information.

Arguments

$color_default
optional
Default value is $tycolor-link-default
The default button color; must be rgb, named color, or hexidecimal
$color_active
optional
Default value is $tycolor-link-active
The hovered and active button color; must be rgb, named color, or hexidecimal
$border_radius
optional
Default value is $tyfy-radius-border
The border radius of the button

Examples

tag.button {
  @include button;
}
a.button[href] {
  @include button(red,blue,0px);
}

@mixin button-outline

Use this mixin to display an HTML element as an outlined button. In the default state, the button background color is transparent and the button text color is the default color. In the active state, the button background color is the active color and the text color is either black or white – whichever provides the best contrast with the background color. To apply the mixin to a link, use the [href] attribute selector. See the links and utility documetation for more information.

Arguments

$color_default
optional
Default value is $tycolor-link-default
The default button color; must be rgb, named color, or hexidecimal
$color_active
optional
Default value is $tycolor-link-active
The hovered and active button color; must be rgb, named color, or hexidecimal
$border_radius
optional
Default value is $tyfy-radius-border
The border radius of the button

Examples

tag.button-outline {
  @include button-outline;
}
a.button-outline[href] {
  @include button-outline(red,blue,0px);
}

@mixin clearfix

Use this mixin to clear floats within an element. This mixin does not accept any arguments. Code comes from HTML5 Boilerplate.

Example

tag.class {
  @include clearfix;
}

@mixin fade-in

Use this mixin to apply a non-looping fade in animation to an element with the given animation duration.

Arguments

$duration
optional
Default value is 1000ms
The duration of the animation in seconds or milliseconds

Example

tag.class {
  @include fade-in(2s);
}

@mixin set-sibling-margins

Use this mixin to set the size of adjacent sibling margins for all children of the selected elements. For more information, see the Margins documentation.

Arguments

$context
optional
Default value is all
Allowed values are all, headings, or lists. The all context sets the value for all tags; the headings context sets the value only for headings; the lists context sets the value only for children of ordered, unordered, and definition lists.
$measure
optional
Default value is $tyfy-spacing-standard
A rem-unit size of the adjacent sibling margins to apply in the given context

Example

tag.class {
  @include set-sibling-margins(all,tygrid(1));
}

@mixin sibling-margins

Use this mixin to turn adjacent sibling margins on or off for children of the selected elements. If the state is on, the margins are set to the Tyfy default values. To override with specific values, use the set-sibling-margins mixin.

Arguments

$state
optional
Default value is on
Allowed values are on or off

Example

tag.class {
  @include sibling-margins(off);
}

@mixin visuallyhidden

Use this mixin to visually hide an element, but allow screen readers and search engines to read the element’s content. The default true argument hides the element. The false argument is used to reverse the default styles. Code is adapted from HTML5 Boilerplate

Arguments

$state
optional
Default value is true
Allowed values are true or false

Example

tag.hide-me {
  @include visuallyhidden;
  &.show-me {
    @include visuallyhidden(false);
  }
}
