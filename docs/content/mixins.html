Mixins

Tyfy includes several SASS mixins that facilitate the use of the framework and provide commonly used styles and effects.

@mixin clearfix

Clear floats within an element. This mixin does not accept any arguments. Code comes from HTML5 Boilerplate.

Example

tag.class {
  @include clearfix;
}

@mixin visuallyhidden

Visually hide an element, but allow screen readers and search engines to read the elementâ€™s content. The default true argument hides the element. The false argument is used to override the default styles. Code is adapted from HTML5 Boilerplate

Arguments

$state
Allowed values are true or false
Default value is true

Example

tag.hide-me {
  @include visuallyhidden;
  &.show-me {
    @include visuallyhidden(false);

@mixin set-sibling-margins ($context:all, $measure:$tyfy-spacing-standard)
A mixin for setting the value of the vertical margin between adjacent siblings. The all context employs the lobotomized owl selector for margins between all elements. The headings context applies margin between a heading and a preceding non-heading element. The lists context applies margin between child element of definition, ordered, and unordered lists.
@mixin sibling-margins ($state:on)
A mixin for turning the vertical margin between adjacent siblings on an off. It depends on the set-sibling-margins mixin.
@mixin fade-in($duration:1000ms)
An effect that fades in the element to which the mixin is applied. The mixin accepts a duration as an argument; the default is 1000ms.
@mixin button($color_default:$tyfy-color-link-default,$color_active:$tyfy-color-link-active,$border_radius:tygrid(0.25))
A mixin for displaying a link or other tag as a button.
@mixin button-outline($color_default:$tyfy-color-link-default,$color_active:$tyfy-color-link-active,$border_radius:tygrid(0.25))
A mixin for displaying a link or other tag as an outlined button.
