Icons

Because Tyfy only needs a handful of icons, I've included a dozen or so common icons as a custom web font. Node.js and the webfont package are used to convert a set of .svg icons into an icon webfont called tyfyicons. The Sketch file used to create the .svgs is included in the Tyfy repository.

Do I Need This Font?

The tyfyicons font is not required. If you want to use a different icon font, customize the values for the icon character code variables in the required/settings module to use character values for your preferred font. Icons are used in styles for:

  • lists
    • $tyfyicons-angle-right
    • $tyfyicons-dash
    • $tyfyicons-check
  • blockquotes
    • $tyfyicons-quote-right
  • collapsables
    • $tyfyicons-angle-down
    • $tyfyicons-angle-up
  • tabs
    • $tyfyicons-angle-down
    • $tyfyicons-angle-up

If you are not planning to use those modules, Tyfy will not need icons and the character code variables ignored.

Using Icons in SCSS

Because of an issue with SASS, the icon font character codes are stored in the required/settings without their preceding slash:

$tyfyicons-angle-up:"ea08";

To properly apply the character codes in the CSS content property, Tyfy has a custom function named charcode that accepts a character code variable and returns the correct character code string.

Using Icons in HTML

The optional/icons module includes utility classes for adding icons within the html. Those utility classes are employed in the following table.

Icon SASS variable Character code
$tyfyicons-angle-up\ea08
$tyfyicons-angle-down\ea01
$tyfyicons-angle-left\ea02
$tyfyicons-angle-right\ea03
$tyfyicons-angle-up-down\ea04
$tyfyicons-bars\ea07
$tyfyicons-x\ea0c
$tyfyicons-check\ea05
$tyfyicons-dash\ea06
$tyfyicons-quote-left\ea0a
$tyfyicons-quote-right\ea0b
$tyfyicons-search\ea09
