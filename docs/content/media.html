Tyfy provides basic styles for images and media embedded using the <iframe> tag — such as YouTube videos. The styles address a few common layout situations. Percentage widths are used to make media responsive.

Image Lazy Loading

Tyfy includes JavaScript (dist/script/lazyimage.js) for the lazy loading of images. Lazy loading of images is triggered by the use of the data-lazyimage attribute on the <img> tag. The data-lazyimage attribute defines the URI of the image to be loaded, and the src attribute is used for a placeholder. The loading of the image is triggered when the <img> appears in the viewport as determined by the Intersection Observer API.

Variables

$tybreak-floats
40em !default;
Screen width above which float styles are enabled.
$tycolor-text-light
mix($tycolor-fg,$tycolor-bg,45%) !default;
A light text color variant, used for captions.

Classes

.tymedia
Apply to the element containing a styled <img> or <iframe>.
.tymedia-caption
Apply to text serving as a caption inside the element with the tymedia class.
.tymedia-iframe
Apply to the element containing an <iframe> element; this must be nested inside the element with the tymedia class.
.float-left, .float-right
Applied to the element with the tymedia class to float the element left or right. At screen widths below the value of $tyfy-break-floats, the float styles are disabled.
