Required

The required modules are the necessary parts of the framework. They provided variables and functions to describe and implement the modular typographic scale and grid. They also describe the style attributes of the framework and provide useful tools for implementing the framework. The four required modules are:

  • _core.scss
  • _functions.scss
  • _settings.scss
  • _mixins.scss
