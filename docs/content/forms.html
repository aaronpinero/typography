Forms

Tyfy provides basic styles for form elements. The styles are intended for simple forms where each form element consists of a label-input pair.

<form>
  <div>
    <label>Text Input</label>
    <input type="text" name="name">
  </div>  
  <div>
    <label>Number Input</label>
    <input type="number" name="age">
  </div>
</form>

Element spacing has been adjusted so labels are more closely positioned to their associated input than to surrounding elements. There are CSS classes for select and label elements that provide some presentation options.

Variables

$tyfont-input
$tyfont-sans !default;
Typeface for form elements.
$tywidth-border
2px !default;
Thickness of the borders around form elements.
$tycolor-border
mix($tycolor-fg,$tycolor-bg,12%) !default;
Default border color.

Classes

.tyselect
Adjusts the appearance of a select element so it is more consistent with other elements; this class can be applied directly to the select element or to a parent element.
.label-option
Apply to a label when used adjacent to or to encapsulate a radio button or checkbox input.
.label-inline
Apply to a label to display as an inline-block instead of block element.

Examples

Inputs

<form>
  <div>
    <label>Text Input</label>
    <input type="text" name="name">
  </div>
  <div>
    <label>Number Input</label>
    <input type="number" name="age">
  </div>
  <p><input type="submit" name="submit" value="Submit Button"></p>
</form>

Selects

<form>
  <div>
    <label>Select</label>
    <div class="tyselect">
      <select name="color">
        <option value="red">Red</option>
        <option value="orange">Orange</option>
        <option value="yellow">Yellow</option>
        <option value="green">Green</option>
        <option value="blue">Blue</option>
        <option value="purple">Purple</option>
      </select>
    </div>
  </div>
  <div>
    <label>Multiple Select</label>
    <div class="tyselect">
      <select name="color" multiple="multiple">
        <option value="red">Red</option>
        <option value="orange">Orange</option>
        <option value="yellow">Yellow</option>
        <option value="green">Green</option>
        <option value="blue">Blue</option>
        <option value="purple">Purple</option>
      </select>
    </div>
  </div>
  <p><input type="submit" name="submit" value="Submit Button"></p>
</form>
