Build Tools

This project uses node.js for assembling the icon webfont, compiling the SCSS files into CSS, and for assembling the documentation pages. These tasks are automated with npm scripts.

> npm run build-css
> npm run watch-css
Uses node-sass to create compressed CSS files. Autoprefixing is accomplished with autoprefixer and postcss-cli. The watch script uses nodemon to check for changes to the SCSS files.
> npm run build-css-dev
> npm run watch-css-dev
Uses node-sass to create readable CSS with source file comments. Files are linted prior to compilation using stylelint. Autoprefixing is accomplished with autoprefixer and postcss-cli. The watch script uses nodemon to check for changes to the SCSS files.
> npm run build-icons
Uses webfont the webfont and SCSS files for the icons.
> node docs.js
Compiles documentation files using templatesjs to assemble the HTML pages and their includes.
