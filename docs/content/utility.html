Utility Styles

Tyfy includes customizations for other block and inline HTML elements. There are also additional classes for styles that do not have an HTML equivalent.

Demo

Bold text Bold text, applied using the <b> or <strong> tags
Italic text Italic text, applied using the <em> or <i> tags
Code Text describing code, using the <code> tag
Code, no background A version of <code> tag using the code-nobg class 
Preformatted
 Preformatted text, using the <pre> tag
Small Small text, using the <small> tag; makes text one scale step smaller
Horizontal rule using the <hr> tag; the thickness is based on the value of the $tyfy-width-border variable
Small Caps Small caps text using the <span> tag with the smallcaps class
Button Text as a button using <span> tag with the button class; also works with other tags
Outlined Text as a outlined button using <span> tag with the button-outline class; also works with other tags
Drop cap A decorative drop cap using the <span> tag with the dropcap class
Sans-serif font Display text using the font indicated by the $tyfy-sans variable by applying the font-sans class
Serif font Display text using the font indicated by the $tyfy-serif variable by applying the font-serif class
Monospace font Display text using the font indicated by the $tyfy-mono variable by applying the font-mono class
