Spacing

The spacing module removes the default top and bottom margins for all HTML elements, replacing those styles with Tyfys adjacent sibling margin rules. The adjacent sibling margin rules are applied if the value of the $tymargins-on setting is true. If you wish to apply the adjacent sibling margins only in specific contexts, you can set the value to false and use the mixins or utility classes to selectively activate the margins.

Variables

$tymargins-on
true !default;
If true, adjacent sibling selectors will be applied to all elements; if false, the default top and bottom margin of all elements is zero.
$tyspacing-md
tygrid(map-get($tyspacing,"medium")) !default;
The default adjacent sibling margin size; adjacent sibling margins between elements in a list will be half this value.
$tyspacing-lg
tygrid(map-get($tyspacing,"large")) !default;
The default top margin for a heading if preceded by a non-heading element.

Classes

The following classes are defined by the spacing module. They can be used to enable or disable the adjacent sibling margins in specific contexts. Please note: these classes may not behave as expected if nested. So, it is not recommended to use the .tymargins-on class on a descendant of an element that is using the .tymargins-off, or vice versa. This is because of how CSS interprets the wildcard selector.

.tymargins-on
Apply the default adjacent sibling margin rules to descendants of the element with this class
.tymargins-off
Remove the default adjacent sibling margin rules from descendants of the element with this class
