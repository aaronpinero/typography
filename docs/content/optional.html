Optional Modules

The optional modules provide styles for content elements and UI patterns. They can be used individually and are not required for the framework. The ten optional modules are:

  • Paragraphs
  • Headings
  • Links
  • Lists
  • Utility
  • Quotes
  • Tables
  • Collapsables
  • Tabs
  • Forms
