Functions

Tyfy includes custom SASS functions that make the framework easier to use.

@function tylead

This function calculates a grid-based line height for a given font size and (optional) line height style. This function helps maintain the baseline grid.

Arguments

$font-size
required
Font size value in rem units
$leading-style
optional
Default value is base
Accepted values are base, heading, tall, or short

Examples

p {
  font-size:1rem;
  line-height:tylead(1rem);
}
p.special {
  font-size:1.2rem;
  line-height:tylead(1.2rem,tall);
}

@function tygrid

This function calculates a grid-based length value. This function facilitates the use of the $tyfy-grid-unit variable by making your SCSS easier to read, write, and maintain.

Arguments

$multiplier
optional
default value is 1
A floating point value representing the number of grid units in the desired size

Examples

div {
  margin-bottom:tygrid(1.5); // 1.5 times the default grid unit length
  width:tygrid(50); // 50 times the default grid unit length
}

@function tyscale

This function facilitates use of the modular typographic scale by automating the calculation of font sizes. The function accepts an integer and returns a font size, in rem units, that is the specified number of steps above or below the base font size on the typographic scale. The value 0 will return the base font size. If the argument is 2, the function will return the font size that is 2 steps larger on the scale. If the argument is -1, the function will return the font size that is 1 step smaller on the scale.

Arguments

$exponent
required
Integer representing a number of steps from the base font size on the modular scale to calculate; positive and negative values allowed.

Examples

$tyfy-font-size-base: 1rem;
small {
  font-size: tyscale(-1) !default; // 0.833rem; one scale step down from the base size
}
h6 {
  font-size: tyscale(0) !default;  // 1rem; using 0 as the exponent is the same as setting the value to the base size
}
h5 {
  font-size: tyscale(1) !default;  // 1.2rem; one scale step up from the base size
}

@function tychar

This function facilitates the use of the icon font by working around a problem in SASS related to character codes. This function properly applies icons when used in the content attribute of a pseudo-element.

Arguments

$code
required
A character code without the preceding slash (“\”). For use with the tyfyicons, there are SASS variables for each of the character codes.

Examples

div::before {
  content: tychar($tyfyicons-angle-left);
}
