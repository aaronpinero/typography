Extras Modules

Extras modules provide enhancements and functionality not targeted to a specific HTML element or UI pattern. Their use is situational and therefore not necessary for every project. If useful for your project, they can be @included in your SCSS file after the other Tyfy modules. The available extras are:

  • Effects
  • Icons
  • Responsive Text
  • Reverse Colors
