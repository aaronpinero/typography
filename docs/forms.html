Forms

Tyfy provides basic styles for form elements. There are helper classes provided for styling select elements and labels that are used in conjunction with checkboxes and radio buttons.

Classes

.tyfy-select
Apply to a div that contains a select element.
.label-option
Apply to a label when used adjacent to or to encapsulate a radio button or checkbox input.
.label-inline
Apply to a label to display as an inline-block instead of block element.
