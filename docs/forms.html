Forms

Tyfy provides basic styles for form elements. The styles are intended for simple, well structured forms where each form element consists of a label-input pair. Standard element spacing has been adjusted so labels are visually associated with the corresponding input. There are helper classes for select and label elements that provide some presentation options.

Classes

.tyfy-select
Apply to a div that contains a select element.
.label-option
Apply to a label when used adjacent to or to encapsulate a radio button or checkbox input.
.label-inline
Apply to a label to display as an inline-block instead of block element.

Variables

$tyfy-width-border: 2px !default;
Width of the border around inputs, styled select boxes, and fieldsets
$tyfy-color-border: mix($tyfy-color-text-body,$tyfy-color-background,12.5%) !default;
Color of the borders around inputs, styled select boxes, and fieldsets
$tyfy-color-link-active: #cc3380 !default;
Color of the border around inputs and styled select boxes that have the input focus

Examples

Text Inputs

<form>
  <div>
    <label>Name</label>
    <input type="text" name="name">
  </div>
  <div>
    <label>Age</label>
    <input type="number" name="age">
  </div>
</form>

Select Boxes

<form>
  <div>
    <label>Favorite Color</label>
    <select name="color">
      <option value="red">Red</option>
      <option value="orange">Orange</option>
      <option value="yellow">Yellow</option>
      <option value="green">Green</option>
      <option value="blue">Blue</option>
      <option value="purple">Purple</option>
    </select>
  </div>
  <div>
    <label>Favorite Animal</label>
    <select name="color" class="tyfy-select">
      <option value="lion">Lion</option>
      <option value="zebra">Zebra</option>
      <option value="monkey">Monkey</option>
      <option value="hippo">Hippo</option>
      <option value="elephant">Elephant</option>
      <option value="rabbit">Rabbit</option>
    </select>
  </div>
</form>
