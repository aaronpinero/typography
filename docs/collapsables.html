Collapsable Headings

Collapsable headings are my implementation of an accordion pattern. Content authors can apply the pattern using single CSS class applied to a heading (level 2 through 6). The associated JavaScript encapsulates all of the content belonging to that heading in a block element and applies a click event to the heading that toggles the display of the content.

Collapsable headings can be nested, and the style varies with the heading level to emphasize the content hierarchy. Additional JavaScript automatically opens the correct collapsable areas in the case where an anchor link is referenced in the window.location.href string.
