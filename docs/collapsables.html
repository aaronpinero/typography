Collapsable Headings

Collapsables

Collapsable headings are the Tyfy implementation of the accordion pattern.

Content authors can apply the pattern using single CSS class applied to a heading (level 2 through 6). The associated JavaScript encapsulates all of the content belonging to that heading in a block element and binds a click event to the heading that toggles the display of the content.

Features

  • Collapsable headings are responsive. At small screen widths, the design changes to maximize horizontal space.
  • Collapsable headings can be nested. Semantically correct heading levels must be used for nesting to work correctly.
  • Collapsable headings accommodate linking to anchors. The correct collapsable section will be opened to show the target of an anchor link.

HTML Structure

Collapsable headings are defined by adding the collaspable class to a heading (H2 through H6). JavaScript is used to identify the content belonging to the heading. The content of a collapsable section is defined as all the next sibling elements following the collaspable heading element up to the next heading of the same or higher level.

<h2 class="collapsable">Collapsable Heading 1</h2>
<p>[content of the collapsable section]</p>
<p>[content of the collapsable section]</p>

<h2 class="collapsable">Collapsable Heading 2</h2>
<p>[content of the collapsable section]</p>
<ul>[content of the collapsable section]</ul>

Variables

The following variables are defined in the required settings module.

$tyfy-break-collapsables: 50em !default;
Breakpoint below which the design changes to maximize horizontal space
$tyfy-color-border: mix($tyfy-color-text-body,$tyfy-color-background,12.5%) !default;
Color of the borders around tab labels and tab panels
$tyfy-width-border: 2px !default;
Width of the border around tab labels and tab panels
$tyfyicons-angle-down: "ea01" !default;
The character code for the angle-down icon
$tyfyicons-angle-up: "ea05" !default;
The character code for the angle-up icon
