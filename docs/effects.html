This module creates classes for applying a CSS fade-in animation to an HTML element.

The module started as an experiment with SASS maps. In this case, I have used a sass map to generate several CSS classes that, when applied to an element, cause that element to fade in when the page is loaded. The duration of the fade animation depends on the class used.

To apply a fade in animation of arbitrary duration, there is a mixin that accepts duration as a variable.

For now, this is the only effect available in Tyfy. I may add more effects at a later time. I welcome suggestions for particular effects.

Classes

<tag class="fade-500ms">
Applies a fade in animation of 500ms duration.
<tag class="fade-1s">
Applies a fade in animation of 1000ms duration.
<tag class="fade-2s">
Applies a fade in animation of 2000ms duration.
<tag class="fade-3s">
Applies a fade in animation of 3000ms duration.

Variables

The following variables are defined in the required settings module.

$fades
A SASS map of class names and duration values use to generate the fade-in effect classes.
$fades: (
  "500ms": 500ms,
  "1s": 1000ms,
  "2s": 2000ms,
  "3s": 3000ms
) !default;
