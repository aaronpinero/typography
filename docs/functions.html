Tyfy includes four custom SASS functions to facilitate use of the framework for front-end development.

@function tylead

Calculates a grid-based line height value based on a font size and (optionally) a line height style. This function helps maintain the baseline grid by automating the calculation of appropriate line heights.

Arguments

$font-size
Font size value in rem units.
$leading-style
Accepted value are base, heading, tall, or short.
Default value is base.

Examples

p {
  font-size:1rem;
  line-height:tylead(1rem);
}
p.special {
  font-size:1.2rem;
  line-height:tylead(1.2rem,tall);
}

@function tygrid

Calculates a grid-based length value. This function facilitates the use of the $tyfy-grid-unit variable by making your SCSS easier to read, write, and maintain.

Arguments

$multiplier
A floating point value to be multipled by the grid unit value to calculate the desired length.

Examples

div {
  margin-bottom:tygrid(1.5); // 1.5 times the default grid unit length
  width:tygrid(50); // 50 times the default grid unit length
}

@function tyscale

Facilitates use of the modular typographic scale by automating the calculation of font sizes. The function accepts an integer and returns a font size, in rem units, that is the specified number of steps above or below the base font size on the typographic scale. The value 0 will return the base font size. If the argument is 2, the function will return the font size that is 2 steps larger on the scale. If the argument is -1, the function will return the font size that is 1 step smaller on the scale.

Arguments

$exponent
Integer representing a number of steps from the base font size on the modular scale to calculate; positive and negative values allowed.

Examples

$tyfy-font-size-base: 1rem;
small {
  font-size: tyscale(-1) !default; // 0.833rem; one scale step down from the base size
}
h6 {
  font-size: tyscale(0) !default;  // 1rem; using 0 as the exponent is the same as setting the value to the base size
}
h5 {
  font-size: tyscale(1) !default;  // 1.2rem; one scale step up from the base size
}

@function tychar

Facilitates the use of the tyfyicons custom font by working around a problem in SASS related to character codes. This function properly applies icons when used in the content attribute of a pseudo-element.

Arguments

$code
A character code without the preceding slash (“\”). For use with the tyfyicons, there are SASS variables for each of the character codes.

Examples

div::before {
  content: tychar($tyfyicons-angle-left);
}
