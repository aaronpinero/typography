Tyfy includes several custom SASS functions. These functions were written to facilitate writing the Tyfy SCSS files, but can also be used to help users who are including Tyfy in their web projects.

@function tylead

Calculates a grid-based line height value based on a font size and (optionally) a line height style. This function helps maintain the baseline grid by facilitating the calculation of appropriate line heights.

Arguments

$font-size
Font size value in rem units
$leading-style
Accepted value are base, heading, tall, or short.
Default value is base

Examples

p {
  font-size:1rem;
  line-height:tylead(1rem);
}
p.special {
  font-size:1.2rem;
  line-height:tylead(1.2rem,tall);
}

@function tygrid

Calculates a grid-based length value. This function facilitates the use of the $tyfy-grid-unit variable by making CSS easier to read and write.

Arguments

$multiplier
A floating point value to be multipled by the grid unit value to calculate the desired length.

Examples

div {
  margin-bottom:tygrid(1.5); // 1.5 times the default grid unit length
  width:tygrid(50); // 50 times the default grid unit length
}
@function tygrid($multiplier)
Makes the SCSS files easier to read. The function accepts a number and returns a value, in rem units, that is the base grid unit multiplied by the number argument.
@function tyscale($exponent)
Facilitates use of the modular typographic scale by eliminating the task of calculating font sizes. The function accepts an integer and returns a font size, in rem units, that is the specified number of steps above or below the base font size on the typographic scale. The value 0 will return the base font size. If the argument is 2, the function will return the font size that is 2 steps larger on the scale. If the argument is -1, the function will return the font size that is 1 step smaller on the scale.
@function tycolor($color,$lightness)
Accepts a color (in rgb or hexadecimal format) and a lightness value (0% to 100%), and returns the given color converted to the specified lightness.
@function tyreverse($color)
Accepts a color (in rgb or hexadecimal format) and returns the same color with the lightness value reversed.
@function tychar($code)
Accepts a character code without the preceding slash (“\”) and adds it; this is done to get around a problem in SASS and allow icon character codes to be more easily customized for different icon fonts.
