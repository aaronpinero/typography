Why Tyfy?

Modular Scale

Tyfy provides variables and SASS functions for setting a modular scale. A modular scale creates typographic contrast and harmonious font size proportions.

Baseline Grid

Tyfy helps you set and maintain a baseline grid. The framework provides a core grid unit variable and SASS functions for calculating grid-based measures.

Sensible Margins

You usually don't need margins around HTML elements, only between HTMl elements. Tyfy provides mixins and classes that help you apply sensible margins.

About Tyfy

Tyfy (short for typography or typogrify) started as – and to some degree, remains – a personal project, a summary of past work, and resumé filler. It is inspired by the writings of Robert Bringhurst and the work of designers like Matej Latin and Richard Rutter.

I also have a related project, Tyfy Bootstrap, that applies elements of this framework as a theme for Bootstrap.
